Interactive Elements

Navigation Button

CSS selector:

.nav-menu-button

Sample code:

<a href="demo-reel.html" class="nav-menu-button nav-demo-reel">Demo Reel</a>

Rendered element:

Demo Reels

In-text link

CSS selector:

.intext-link

Sample code:

<a href="#" class="intext-link">Default Link</a>

Rendered element:

This is a text based link: Default Link

Submit/Reset Buttons

CSS selector:

.button

Sample code:

<a href="#" class="button-submit">Default Button</a>

Rendered element:

Submit

Text Input Field

CSS selector:

.text-field

Sample code:

<input class="text-field" type=text name="name" value="Enter Name"></input>

Rendered element:

Text Elements

Headings

CSS selector:

h1, h2, h3

Sample code:

<h3>Heading 3</h3>

Rendered element:

Heading 3

Bulleted Lists

CSS selector:

ul

Sample code:

<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Rendered element:

  • Item 1
  • Item 2
  • Item 3

Numbered Lists

CSS selector:

ol

Sample code:

<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>

Rendered element:

  1. Item 1

  3. Item 3
